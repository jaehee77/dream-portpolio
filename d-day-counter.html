Web Storage

Session Storage

  • Key-value 형태로 저장
  • 로컬 환경에 데이터를 저장
  • 세션 단위로 구분되며 활용
    • 세션 : 사용자가 브라우저를 통해 웹 페이지에 접속한 시점부터, 해당 웹페이지 접속을 종료하는 시점까지를 의미하는 단위
    • 즉, 사용자의 웹페이지 접속 상태에 따른 단위를 말한다.
  • 브라우저, 탭을 종료하면 데이터 영구 삭제

Local Storage

  • Key-value 형태로 저장
  • 로컬 환경에 데이터를 저장
  • 도메인 단위로 구분되며 활용
    • http://localhost:5500/user
      /user 인 엔드포인트가 달라져도 같은 도메인이면 데이터를 공유할 수 있음
    • http://localhost:5500/category
  • 브라우저 자체를 종료해도 데이터가 존재
